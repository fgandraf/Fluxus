# Fluxus

Software para gestão de Ordens de Serviços emitidas pela Caixa Econômica Federal às empresas de Engenharia e Arquitetura credenciadas em edital públicos.

---

## Tecnologias

* .NET 6;
* Windows Forms;
* C#;
* Microsoft.Office.Interop.Excel;
* iTextSharp;
* Newtonsoft;
* RestAPI.
